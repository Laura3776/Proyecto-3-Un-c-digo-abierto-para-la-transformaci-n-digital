SmartLog Monitor
Inicio Equipo

Equipo y enfoque

Este trabajo fue planteado para demostrar la aplicacion practica de conceptos de transformacion digital en un software funcional.

Objetivo academico

Implementar una solucion modular que gestione eventos y evidencie buenas practicas de ciclo de vida del dato y seguridad.

Escalabilidad

La arquitectura permite integrar servicios cloud e IoT para futuras versiones.

Temas clave desarrollados

  • Almacenamiento local con SQLite.
  • Integridad y trazabilidad de datos.
  • Modularidad del codigo para mantenimiento.
  • Preparacion para entornos IT y OT.

Proyecto 3 - Un codigo abierto para la transformacion digital.

export loadIncidents () → Array<Object>

Loads all incidents from localStorage. Returns an empty array if storage is empty or if JSON parsing fails.

Returns
Array<Object>
Array of incident objects. Empty array on error or no data.
Example
const incidents = loadIncidents();
console.log(incidents.length); // 12

export validateIncidentInput (formData: FormData) → Object

Validates incident form data submitted by a reporter. Checks required fields, minimum lengths, and allowed enum values. Returns on the first error found.

Parameters
formData
FormData
Raw FormData from the incident creation form.
Returns
Object
{ ok: true, value } on success · { ok: false, message } on error.
Example
const result = validateIncidentInput(formData);
if (!result.ok) {
  showError(result.message);
} else {
  createIncident(result.value);
}
export normalizeImportedIncident (raw: Object) → Object | null

Normalizes a raw imported incident object. Applies defaults for optional fields and returns null for invalid required fields — callers silently skip nulls during bulk import.

main

UI utility functions. Handles navigation menu toggling and copy-to-clipboard. Loaded on all application pages.

scripts/main.js
Behaviours
listener Mobile navigation toggle #menuBtn click → #mainNav.open

Toggles the open CSS class on #mainNav when #menuBtn is clicked. Does nothing if either element is absent from the DOM.

listener Copy-to-clipboard buttons .copy-btn click → clipboard

Initializes copy behaviour for all .copy-btn elements. Reads the text from data-copy. Shows "Copiado" for 1.2 s as feedback. Silently fails if Clipboard API is unavailable.

HTML Usage
<button class="copy-btn" data-copy="npm install">Copy</button>

function buildToc (tab: string) → Array<HTMLAnchorElement>

Builds TOC anchor links in #docToc for every <h2> found in panels matching the given tab. Returns empty array if no container is found.

Parameters

string
Tab identifier, e.g. "funcional".
Returns

function applyTab (tab: string) → void

Switches the active documentation tab. Updates button states, shows/hides panels, rebuilds TOC, and scrolls to top.

Example

function setActiveLink () → void

Updates the active TOC link based on scroll position. Highlights the link whose section is nearest the top of the viewport. Called on every scroll event and on load.
